Web Development Diary

Day 01

Today I decided to signup for the Front End Web Development Treehouse Techdegree. I watched a course called Introduction to HTML and CSS by Treasure Porth. The course is an excellent start to give us a foundation about the structure of the web pages.

I've learned that we use HTML - HyperText Markup Language - to structure the content of our pages. Also, I learned that we use CSS - Cascading Style Sheets - to give instructions to the browser about how to show our content, making our pages beautiful and attractive to the visitors.

After every topic of the course, we have a questionnaire about what we've studied and a coding test that makes us seating the hands on the keyboard and indeed code a few lines.

I'm very excited about what is coming next!

Day 02

Today I finished the second course in Unit 1, HTML Basics. I watched the videos, did all the tests and coding challenges. The teacher is Guil Hernandez.

I've learned a lot about semantic HTML, that add meaning for the sections in our pages. In semantic HTML, we use self-explained tags like:

            <header>Container for introductory content and/or navigation menu</header>
            <section>Creates a new section in your page</section>
            <main>Represents the fundamental content of your page</main>
            <footer>Container for aditional information such as Copyright, Contact, Aditional Navigation, Authorship, etc.</footer>

As you can see, Semantic HTML is self-explained and makes the code understandable for humans, search engines and browsers.
